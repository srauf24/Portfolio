Website Screenshot Gallery

GitHub Repo  |  Live Demo

A React application that allows users to capture and store screenshots of any website using the Apiflash API. Users can specify screenshot format (JPEG/PNG/WebP), dimensions, and block ads or cookie banners for a cleaner image. The app displays generated images in a built-in gallery, providing quick access and reference.

Key Features:

  • Customizable screenshot settings (format, width, height, no_ads, no_cookie_banners).
  • On-demand API call to Apiflash for real-time capture of any public URL.
  • Automatic URL protocol handling (https:// prefix if missing).
  • Screenshot gallery to display previously captured images, stored in state.

Technologies:

  • React (Create React App) for a component-based front end.
  • Apiflash API for on-demand website screenshot capture.
  • CSS3 for a modern, responsive UI with gradients and hover effects.
  • JavaScript for handling events, form input, and state management.
  • Vercel for fast deployment and hosting.
